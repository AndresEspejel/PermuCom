# PermuCom

Proyecto GRADLE escrito en Java para crear Combinaciones y permutaciones a base de un CSV de entrada
 
 ## iniciar proyecto
 compilar poryecto  
    
    gradle build
    gradle test
 
 ## como usarlo
 el archivo que va a leer el programa debe estar en la ruta  
 
    PermuCom\src\main\resources\csv\input
